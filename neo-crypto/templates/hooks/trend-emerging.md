# 📈 TREND EMERGING TEMPLATE

## Hook Variations

### Type 1: Narrative Shift
```
🌊 TIDE IS TURNING

The [Narrative] narrative is gaining momentum

• Volume: Up [Volume]% in 24h
• Social mentions: [Mentions]% increase
• New wallets: [Count] entering
• Key tokens: [Token1], [Token2], [Token3]

This could be the next [Comparison].
```

### Type 2: Sector Rotation
```
🔄 SECTOR ROTATION DETECTED

Money is flowing from [Old Sector] to [New Sector]

• Outflows: $[Amount] from [Old Sector]
• Inflows: $[Amount] to [New Sector]
• Leading token: [Token] up [Gain]%
• Timeframe: Last [Time Period]

Whales are repositioning for [Reason].
```

### Type 3: Early Signal
```
🔍 EARLY SIGNAL SPOTTED

[Indicator] is flashing for [Sector/Token]

• Metric: [Metric Name]
• Current value: [Value]
• Historical threshold: [Threshold]
• Accuracy: [Accuracy]% in past

Last time this happened, [Token] pumped [Gain]%.
```

### Type 4: Contrarian Play
```
🎯 CONTRARIAN OPPORTUNITY

Everyone is focused on [Popular Thing], but [Overlooked Thing] is setting up

• Sentiment: [Sentiment Level] bearish
• Fundamentals: [Fundamental Strength]
• Technicals: [Technical Setup]
• Catalyst: [Upcoming Catalyst]

The crowd is wrong about [Reason].
```

## Template Variables

- `[Narrative]` - Emerging narrative/story
- `[Volume]` - Percentage volume increase
- `[Mentions]` - Social media mention increase
- `[Count]` - Number of new wallets
- `[Token1]` - Example token 1
- `[Token2]` - Example token 2  
- `[Token3]` - Example token 3
- `[Comparison]` - Historical comparison
- `[Old Sector]` - Sector losing money
- `[New Sector]` - Sector gaining money
- `[Amount]` - Dollar amount
- `[Token]` - Leading token in sector
- `[Gain]` - Percentage gain
- `[Time Period]` - Timeframe (e.g., "48 hours")
- `[Reason]` - Reason for rotation
- `[Indicator]` - Technical/fundamental indicator
- `[Sector/Token]` - What it's indicating for
- `[Metric Name]` - Name of metric
- `[Value]` - Current metric value
- `[Threshold]` - Historical threshold level
- `[Accuracy]` - Historical accuracy percentage
- `[Popular Thing]` - What everyone is watching
- `[Overlooked Thing]` - What's being ignored
- `[Sentiment Level]` - Market sentiment
- `[Fundamental Strength]` - Fundamental analysis
- `[Technical Setup]` - Technical analysis
- `[Upcoming Catalyst]` - Future catalyst

## Best Practices

1. **Frame as discovery** - "I noticed", "Detected", "Spotted"
2. **Use data points** - Specific numbers build credibility
3. **Add historical context** - "Last time this happened..."
4. **Identify catalysts** - Why is this happening now?
5. **Keep it actionable** - What should people watch?
6. **Add intrigue** - "The crowd is wrong about..."
