## 🚀 QUICK START

### 1. Installation
```bash
# Clone the skill
git clone https://github.com/iseeiape/neo-crypto.git

# Install dependencies
cd neo-crypto
npm install
```

### 2. Configuration
```bash
# Copy example config
cp config/neo-config.example.json config/neo-config.json

# Edit with your API keys
nano config/neo-config.json
```

### 3. Run
```bash
# Start the content engine
node scripts/alpha-generator.js

# Or run the full pipeline
./run.sh
```

## 🎭 CONTENT FRAMEWORK

### 6-Step Crypto Content Formula

**1. ATTENTION (Hook)**
> "This wallet turned $1k into $2.4M in 48 hours"

**2. PAIN (Problem)**
> "Most traders miss these early signals"

**3. FIND (Discovery)**
> "I tracked 47 whales and found the pattern"

**4. SETUP (Transformation 1)**
> "Here's what they bought and when"

**5. RESULTS (Transformation 2)**

## 💡 PRO TIPS

### Content Strategy
1. **Volume beats perfection** - 3 good posts/day > 1 perfect post/week
2. **Be early, not right** - Share signals as you find them
3. **Show the process** - People love behind-the-scenes
4. **Engage, don't broadcast** - Reply to comments, build community

### Technical Tips
1. **Use cheap models** for routine content generation
2. **Batch process** - Generate a week's content at once
3. **A/B test** hooks and CTAs
4. **Track everything** - Data informs improvement
